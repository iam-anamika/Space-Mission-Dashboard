## 📌 Project Overview

The **Space Mission Dashboard** is an interactive Power BI project that analyzes historical space mission data through meaningful visualizations and KPIs. The dashboard provides insights into launch trends, mission outcomes, space agencies, rocket status, and launch locations, enabling users to explore the history of space exploration interactively.


## ✨ Features

- 🚀 Total Launches KPI
- 📈 Mission Success Rate
- 🛰️ Active Rockets
- 🏢 Total Space Agencies
- 🌍 Interactive Map of Launch Locations
- 📊 Top Organizations by Launch Count
- 📉 Launch Trend Over the Years
- 🍩 Mission Status Distribution
- 🎛️ Interactive Filters (Mission Status, Rocket Status, Organization)

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization


## 📂 Dataset

The dataset contains historical records of space missions with the following information:

- Organization
- Launch Date
- Launch Location
- Mission Details
- Rocket Status
- Mission Status
- Launch Cost (Price)

## 📊 Key Insights

- Most recorded space missions were successful.
- Launch activity has increased significantly over the years.
- A small number of organizations account for the majority of launches.
- Space missions are concentrated at a few major launch sites worldwide.
- Interactive filters allow users to explore data by organization, mission status, and rocket status.
